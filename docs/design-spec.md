## 字体

- 中文：系统默认中文字体 (PingFang SC, Microsoft YaHei)
- 英文：Inter
- 代码/术语等宽：JetBrains Mono

## 布局

```
┌──────────┬─────────────────────────────────┐
│ 左侧导航  │         顶部导航栏               │
│ (固定)   ├─────────────────────────────────┤
│          │                                 │
│ 240px   │         主内容区                  │
│          │        (可变宽)                   │
│          │                                 │
│          │                                 │
│          │                                 │
└──────────┴─────────────────────────────────┘
```

## 组件风格

- 圆角：默认 8px，按钮 6px，卡片 12px
- 阴影：卡片使用浅阴影 (shadow-sm)
- 按钮：主按钮实心、次按钮描边、文字按钮无背景
- 输入框：描边输入框，聚焦时主色边框
- 弹窗：居中模态框，背景半透明遮罩

## 响应式断点

- 移动端：< 768px（侧边栏折叠为汉堡菜单）
- 平板：768px - 1024px（侧边栏可折叠）
- 桌面：> 1024px（侧边栏常驻）

# 分阶段执行步骤

## 阶段零：项目初始化

1. 创建 CLAUDE.md 项目指引
2. 创建 docs/ 规范文档
3. 创建 dev-logs/ 开发日志目录
4. 初始化 Next.js 项目
5. 配置 TypeScript、Tailwind CSS、shadcn/ui
6. 配置 Prisma + SQLite（开发阶段）

## 阶段一：基础搭建

1. 设计数据库 Schema
2. 配置 NextAuth.js（用户名密码 + 微信）
3. 实现登录/注册页面
4. 搭建左侧导航栏 + 主面板布局
5. 创建所有功能模块空壳页面

## 阶段二：语料库

1. 语料库 CRUD API
2. 公开语料浏览器页面
3. 语料搜索和筛选
4. 批量导入功能
5. 私人语料库管理
6. 公私互通逻辑

## 阶段三：术语库 + 记忆库

1. 术语库 CRUD API 和页面
2. 记忆库 CRUD API 和页面
3. 批量导入术语
4. 跨模块互通

## 阶段四：口笔译练习 + 评分

1. 练习 CRUD API 和页面
2. 双语编辑器
3. 机器评分引擎
4. 社区互评

## 阶段五：团队 + 帖子

1. 团队 CRUD 和邀请
2. 帖子信息流
3. 评论和点赞
4. 通知系统

## 阶段六：实时字幕

1. 音频捕获
2. 语音识别集成
3. 字幕编辑器
4. 翻译引擎切换

## 阶段七：打磨上线

1. 性能优化
2. 安全加固
3. 部署上线
4. 域名绑定

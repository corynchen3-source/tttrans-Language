# 需求文档

## 项目背景

构建一个面向翻译从业者和学习者的中英翻译综合社区网站，用户既可以在此进行翻译练习和语料积累，也可以共享翻译资源、互相交流。

## 目标用户

- 翻译专业学生
- 职业译员
- 语言学习者
- 翻译团队/公司

## 功能需求

### 1. 用户系统

- 注册方式：用户名 + 密码、微信扫码登录
- 个人主页：头像、简介、翻译统计
- 隐私设置：默认内容可见范围

### 2. 语料库

- **公开语料素材库**：所有用户可浏览、搜索、上传双语语料
- **私人语料库**：每个用户可创建多个不同类型的私人语料库
- **公私互通**：公开库语料可一键导入私人库，自动查重补缺
- **批量导入**：支持 TMX、CSV、Excel 格式文件导入
- **可见性**：私密 / 团队可见 / 所有人可见

### 3. 记忆库

- 用户满意的译文存放处
- 分类管理、搜索
- 可选来源追溯（来自语料库/练习/手动输入）
- 可见性控制

### 4. 术语库

- 术语双语对照条目的增删改查
- 批量导入（CSV、TBX格式）
- 术语分类（按领域：法律、医学、科技等）
- 可见性控制

### 5. 口笔译练习

- 用户上传原文+参考译文（或使用共享资源）
- 双语对照编辑练习
- 机器评分：相似度分数 + 具体批改意见
- 社区互评：其他用户可点评
- 满意译文可存入记忆库或术语库

### 6. 实时字幕翻译

- 浏览器捕获系统音频，实时生成双语字幕
- 上传音视频文件，生成双语字幕
- 翻译引擎可选：Google / 有道 / 百度 / 个人语料库
- 字幕编辑：时间轴调整、文本编辑
- 字幕导出：SRT / VTT / TXT
- 字幕片段可导入语料库

### 7. 团队功能

- 用户可创建团队
- 邀请成员加入（链接/邮箱）
- 团队资源共享
- 角色管理：创建者 / 管理员 / 成员

### 8. 社区帖子

- 主面板信息流
- 帖子类型：原文译文推荐、语料库分享、双语素材、术语分享、翻译讨论
- 评论、点赞
- 通知系统

## 非功能需求

- 支持 1 万+ 用户规模
- 中文界面
- 响应式设计（支持移动端）
- 数据安全保障
- 翻译相关操作低延迟
